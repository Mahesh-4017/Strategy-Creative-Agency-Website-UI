# 🚀 Strategy — Creative Agency Website UI

A modern, high-end **Creative Agency Website UI** built using pure **HTML, CSS, and JavaScript**.
This project focuses on **premium design aesthetics, smooth animations, and responsive layout**.

---

## 🌐 Live Demo

👉 https://clinic-website-ui.netlify.app

---

## 📌 Features

* ✨ Modern dark UI design
* 🎯 Fully responsive layout
* 🎬 Smooth scroll & reveal animations
* 🖱️ Custom animated cursor
* 📊 Interactive sections (FAQ, Services, Portfolio)
* 💼 Clean and structured layout for agencies
* ⚡ Lightweight (No frameworks)

---

## 🧱 Tech Stack

* HTML5
* CSS3 (Custom Properties / Variables)
* Vanilla JavaScript
* Bootstrap Icons (CDN)
